Entrega de datos

Esta API entrega los datos en formato JSON, comenzando con dos vectores: "items" y "video".

{
	"items": {}, 
	"video": {}
}

Dentro de items

"items": {
	"reload": Tiempo en recargar el sitio - String,
	"demora_formacion_1": Tiempo de demora del próximo tren - String,
	"demora_formacion_2": Tiempo de demora del siguiente tren - String,
	"demora_formacion_3": Tiempo de demora del subsiguiente tren - String,
	"estacion_hasta_1": Estación de destino del próximo tren - String,
	"formacion_tipo_servicio_1": Tipo de servicio del próximo, puede ser "N", "S", "R" o "E" - String,
	"estaciones_1": Estaciones donde se detiene el próximo tren, en caso de ser un servicio semirrápido, rápido o especial - String,
	"formacion_tipo_servicio_2": Tipo de servicio del siguiente, puede ser "N", "S", "R" o "E" - String,
	"estaciones_2": Estaciones donde se detiene el siguiente tren, en caso de ser un servicio semirrápido, rápido o especial - String,
	"formacion_tipo_servicio_3": Tipo de servicio del subsiguiente, puede ser "N", "S", "R" o "E" - String,
	"estaciones_3": Estaciones donde se detiene el subsiguiente tren, en caso de ser un servicio semirrápido, rápido o especial - String,
	"alerta": {
		"estado_i": La alerta se mostrará o no sólo en el lado de ida - Valores 0 y 1 - Int,
		"estado_v": La alerta se mostrará o no sólo en el lado de vuelta - Valores 0 y 1 - Int,
		"mensaje": Mensaje de la alerta - String,
		"sentido": Sentido donde se mostrará la alerta - String
	}
}

Dentro de video

"video": {
	"mostrar": Mostrará el video según el valor, 0 o 1 - Int,
	"nombre": Nombre del archivo que cargará como video, generalmente local - String,
	"duracion": Duración en milisegundos del video - String,
	"intervalo": Intervalo de aparición del video en milisegundos - String
}

Plataforma

Esta API entrega los datos en un formato extraño, separado por guiones bajos _.

Estructura de ejemplo

ok_11:50_2_3_11:46_RETIRO - PALERMO - CHACARITA - LA PATERNAL - VILLA DEL PARQUE - DEVOTO - SAENZ PEÑA - SANTOS LUGARES - CASEROS - EL PALOMAR - HURLINGHAM - W.C. MORRIS - BELLA VISTA - MUÑIZ - SAN MIGUEL - JOSE C. PAZ - SOL Y VERDE - PTE. DERQUI - VILLA ASTOLFI - PILAR _0_0_0__

Valores

Item Qué es Qué valor entrega Qué muestra
1 Estado ok o null Muestra si hay un tren en la plataforma o no.
2 Hora actual Hora actual o 0 Muestra la hora actual de la API.
3 Plataforma Número El número de la plataforma.
4 Hora de partida Hora de partida o 0 Muestra la hora de partida del tren.
5 Estaciones Cadena de caracteres Muestra las estaciones donde para la formación.
